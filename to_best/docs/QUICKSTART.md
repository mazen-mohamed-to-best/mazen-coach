## ⚡ الخطوة 1: تثبيت Flutter

### Windows
```powershell
# 1. حمّل Flutter SDK
# https://docs.flutter.dev/get-started/install/windows

# 2. أضف flutter/bin إلى PATH

# 3. تحقق
flutter doctor
```

### macOS
```bash
brew install --cask flutter
flutter doctor
```

### Linux
```bash
sudo snap install flutter --classic
flutter doctor
```

---

## ⚡ الخطوة 2: تثبيت الحزم وتشغيل

```bash
# 1. اذهب إلى مجلد المشروع
cd to_best

# 2. تثبيت الحزم
flutter pub get

# 3. تحقق من الجهاز المتصل
flutter devices

# 4. شغّل التطبيق
flutter run

# أو على جهاز محدد
flutter run -d emulator-5554
```

---

## ⚡ الخطوة 3: إعداد الاتصال بالـ Backend

بعد تشغيل التطبيق:

1. **شاشة تسجيل الدخول** — اضغط "الاتصال" في الأسفل
2. أدخل **رابط WebApp** (من Google Apps Script)
3. أدخل **مفتاح الأمان** (Secret Key)
4. اضغط "اختبار الاتصال"
5. إذا ظهر "تم الاتصال بنجاح ✓" → اضغط "رجوع لتسجيل الدخول"

---

## ⚡ الخطوة 4: تسجيل الدخول

### للمستخدم العادي
1. اضغط "حساب جديد"
2. أدخل البيانات المطلوبة
3. انتظر موافقة المدرب

### للأدمن
```
البريد: admin@tobest.app (أو ما تم تحديده في Apps Script)
كلمة المرور: admin_password
```

## 📱 دليل الميزات

### 🏠 الرئيسية
- عرض تمرين اليوم
- آخر الأرقام القياسية
- إحصائيات سريعة

### 🏋️ التمرين
1. اختر البرنامج (أيقونة ↔ في أعلى اليمين)
2. اضغط على الجلسة المطلوبة
3. ابدأ بـ بروتوكول الإحماء
4. سجّل الأوزان والعدات لكل تمرين
5. استخدم مؤقت الراحة (أيقونة ⏱)
6. اضغط "إنهاء الجلسة" عند الانتهاء

### 🥗 التغذية
1. اضغط "إضافة طعام"
2. أدخل اسم الطعام والكميات
3. حدد نوع الوجبة (إفطار/غداء/عشاء/وجبة خفيفة)
4. تابع تقدمك نحو هدف السعرات
5. سجّل كمية الماء اليومي

### 📅 الحضور
1. اضغط على اليوم في التقويم
2. اختر: حضور ✔ / غياب ✘ / راحة 🛌
3. أو اضغط "سجّل اليوم" للسجل السريع
4. تابع نسبة الإلتزام الشهرية

### 📊 التقدم
- **القياسات**: أضف وزنك وقياساتك الجسمانية
- **الأوزان**: عرض رسوم بيانية لتطور أوزان التمارين
- **الإحصائيات**: إجمالي الجلسات والمجاميع والحجم

### 💬 الشات
- **المجموعة العامة**: الدردشة مع جميع المتدربين
- **الإعلانات**: إعلانات من المدرب (قراءة فقط)
- **المدرب**: (يتطلب اشتراك كامل)
- **AI مساعد**: (يتطلب اشتراك كامل)

للرد: اضغط مطوّلاً على الرسالة → "رد"

### ⚙️ الإعدادات
- غيّر الثيم من الأسفل
- غيّر اللغة (عربي/English)
- اختر لون التمييز
- حدد أيام تمرينك
- غيّر كلمة المرور

## 🔑 كودات الضيف

إذا أعطاك المدرب "كود ضيف":
1. في صفحة تسجيل الدخول، اضغط "دخول كضيف"
2. أدخل الكود
3. ستدخل بصلاحيات محدودة

---

## 💳 الاشتراك

1. اذهب إلى **الإعدادات** → **الاشتراك**
2. اختر الخطة المناسبة
3. اختر المدة (خصومات عند الاشتراك لفترة أطول)
4. أدخل كود الخصم إن وُجد
5. حوّل المبلغ للرقم المحدد
6. ارفع إيصال التحويل
7. اضغط "إرسال الطلب"
8. انتظر موافقة المدرب (عادة 24 ساعة)

## ❓ الأسئلة الشائعة

**س: التطبيق لا يتصل بالـ Backend**
ج: تأكد من رابط WebApp وأن الـ Apps Script منشور بـ "Anyone can access"

**س: البيانات لا تُحفظ**
ج: التطبيق يحتاج إنترنت للمزامنة. يعمل offline لكن يحفظ في قائمة انتظار.

**س: اشتراكي غير مفعّل**
ج: الاشتراك يتطلب موافقة يدوية من المدرب. تواصل معه مباشرة.

**س: كيف أغير البرنامج التدريبي؟**
ج: التمرين → أيقونة ↔ في الأعلى → اختر البرنامج

**س: كيف أضيف تمارين مخصصة؟**
ج: اختر برنامج CUSTOM من قائمة البرامج، ثم أضف جلساتك وتمارينك
